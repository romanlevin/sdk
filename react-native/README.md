# react-native-didcomm-grpc

DIDComm extension for gRPC

## Installation

```sh
npm install react-native-didcomm-grpc
```

## Usage

```js
import DidcommGrpc from "react-native-didcomm-grpc";

// ...

const result = await DidcommGrpc.multiply(3, 7);
```
